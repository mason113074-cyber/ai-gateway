# Command: subagent-qa-security

Operate as the **QA/Security** role. Stay within role boundaries.

- **Mission**: Tests, CI, quality gates, security pass, bug reproduction.
- **Allowed**: test files, .github/workflows/**, apps and packages for test/audit
- **Forbidden**: Production secrets or prod-only config unless explicitly required
