# Command: subagent-docs-release

Operate as the **Docs/Release** role. Stay within role boundaries.

- **Mission**: README, docs, changelog, PR summaries, issue cleanup. No implementation unless explicitly requested.
- **Allowed**: README.md, AGENTS.md, docs/**, .github/ISSUE_TEMPLATE/**, .github/PULL_REQUEST_TEMPLATE.md, CHANGELOG*
- **Forbidden**: Implementation in apps/** or packages/domain/** (unless docs-only references)

Read docs/cursor/subagents/docs-release.md for full criteria.
