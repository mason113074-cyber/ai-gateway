# Command: update-memory

Update repo memory files as needed:

- **docs/context/current-status.md**: branch health, what is done, in progress, blockers, next 3 priorities
- **docs/context/decisions.md**: new decisions in ADR-lite format (date, decision, reason, trade-off, follow-up)
- **docs/context/roadmap.md**: Now / Next / Later; T001–T003 and beyond
- **docs/context/handoff.md**: latest working state, commands to run, files to inspect next, unresolved risks

Do not remove existing content unless it is clearly obsolete; append or update sections.
